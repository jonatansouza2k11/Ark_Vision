```python
def count_people_in_zone(self, zone_name):
    """Count people currently in specific zone"""
    count = 0
    for track_id, state in self.track_state.items():
        if state["zone_idx"] == zone_name and state["status"] == "IN":
            count += 1
    return count
```

================================================================================
                     DICAS E TRUQUES
================================================================================

1. SEMPRE inclua contexto INTEIRO
   - Agente aprende 100x melhor com contexto
   - Vale a pena o tamanho

2. USE ARQUIVO MELHOR PARA SUA PLATAFORMA
   - ChatGPT/Claude: Use .md (mais bonito)
   - Sistema CLI: Use .txt (mais compatível)
   - Automação: Use .yaml (parseável)

3. SEJA ESPECÍFICO NA TAREFA
   - "Faça uma função" = 50% chance de sucesso
   - "Faça function detect_crowding que..." = 95% chance

4. MENCIONE RESTRICOES
   - "Sem GPU" = sugestões otimizadas
   - "Webcam USB" = soluções reais
   - "Windows 10" = compatibility certa

5. PEÇA EXEMPLOS
   - "Me dá código?" = vago
   - "Código exemplo que funcione com yolo.py?" = específico

6. ITERACAO
   - Primeira resposta pode não ser 100% perfeita
   - Peça refinamentos: "Pode fazer mais rápido?"
   - Agente aprende com feedback
